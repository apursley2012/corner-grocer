Corner Grocer

Grocery frequency analysis, enhancement history, and interactive static demo

Project Overview

I originally built Corner Grocer as a C++ console application that reads grocery purchase data, counts how often each item appears, and presents the results as a lookup, frequency list, or histogram.

I later rewrote the application in Python, reorganized it into a class-based structure, added validation and backup features, expanded the sorting and search tools, and connected it to SQLite for persistent storage. This portfolio version turns the completed work into a static browser demo while keeping the logic and purpose of the original application recognizable.
